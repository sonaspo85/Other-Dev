Reminder

Add reminders for events and set alarms outside of the Calendar app.

Create a reminder

Set up and track reminders on your watch:

  1. From Apps, tap Reminder.
  2. Tap Write, and then enter the reminder information.
  3. Tap Set time, set the time for your reminder, and then tap Next.
  4. Set the date and repeat options, then tap Done.
  5. Tap Save when finished.

Complete a reminder

To mark reminders as complete:

  • From Reminder, tap the checkbox beside a reminder to set it as complete.
    • To review completed reminders, tap Completed.
